HTML

O termo "HTML" se refere à Linguagem de Marcação de Hipertexto (do inglês, HyperText Markup Language). Ele é a base da estrutura de qualquer página web que você acessa na internet.

Em termos simples, o HTML é usado para estruturar o conteúdo de uma página. Pense nele como o esqueleto de um corpo humano: ele define onde fica a cabeça, os braços, as pernas, mas não se preocupa com a cor da pele ou o tipo de roupa. Essas características visuais são trabalho do CSS.

O que o HTML faz?

  • Define Títulos e Parágrafos: Você usa tags HTML para indicar que uma parte do texto é um título (<h1>, <h2>, etc.) ou um parágrafo (<p>).
  • Insere Imagens e Vídeos: Com o HTML, você especifica onde uma imagem (<img>) ou um vídeo (<video>) deve aparecer na página.
  • Cria Links: O HTML é fundamental para criar os links (<a>) que conectam diferentes páginas na web. É por isso que é chamado de "hipertexto" – ele permite navegar entre documentos.
  • Organiza Listas e Tabelas: Ele permite estruturar informações em listas (<ul>, <ol>) ou tabelas (<table>).
  • Cria Formulários: Para interagir com o usuário, como em formulários de contato ou login, o HTML fornece os elementos (<form>, <input>, <button>) necessários.

CSS

Cascading Style Sheets (em português, Folhas de Estilo em Cascata). Ele é a linguagem que usamos para estilizar documentos escritos em HTML. Enquanto o HTML define a estrutura e o conteúdo de uma página web (o esqueleto), o CSS é responsável por toda a aparência e pelo layout (a pele, as roupas, a maquiagem, a posição dos elementos).
Pense no CSS como o designer de moda da web. Ele decide:

  • Cores: Quais cores usar para textos, fundos, botões.
  • Fontes: Qual tipo de letra, tamanho e peso (negrito, itálico).
  • Layout: Onde os elementos devem aparecer na página (lado a lado, um embaixo do outro, centralizados).
  • Espaçamento: Quanta margem e preenchimento (padding) deve haver entre os elementos.
  • Tamanhos: O tamanho de caixas, imagens, etc.
  • Animações e Transições: Como os elementos se movem ou mudam de estado.
  • Responsividade: Como a página se adapta a diferentes tamanhos de tela (celulares, tablets, desktops).

Como o CSS Funciona?

O CSS funciona através de regras de estilo. Cada regra consiste em:

  • Seletor: O que você quer estilizar (ex: um parágrafo, um título, um elemento com uma ID específica).
  • Declaração: Como você quer estilizar isso, dentro de chaves { }. Uma declaração tem uma propriedade (o que você quer mudar, ex: color) e um valor (como você quer mudar, ex: blue).

Exemplo Básico:


                /* Este é um comentário em CSS */
                
                /* Seletor: P (todos os parágrafos) */
                p {
                    /* Declarações: */
                    color: #333;        /* Define a cor do texto para um cinza escuro */
                    font-size: 16px;    /* Define o tamanho da fonte para 16 pixels */
                    line-height: 1.5;   /* Define o espaçamento entre as linhas */
                }
                
                /* Seletor: .minha-classe (elementos com a classe "minha-classe") */
                .minha-classe {
                    background-color: lightblue; /* Define a cor de fundo como azul claro */
                    padding: 10px;               /* Adiciona 10 pixels de preenchimento interno */
                    border-radius: 5px;          /* Arredonda os cantos em 5 pixels */
                }
                
                /* Seletor: #meu-id (elemento com a ID "meu-id") */
                #meu-id {
                    text-align: center; /* Centraliza o texto */
                    margin-top: 20px;   /* Adiciona 20 pixels de margem superior */
                }

Onde o CSS é Escrito?

Existem três formas principais de incluir CSS em uma página web:

CSS Externo (Recomendado):

O CSS é escrito em um arquivo separado (.css) e linkado ao HTML usando a tag <link> no <head>.
Vantagem: Organiza o código, facilita a manutenção e o reuso de estilos em múltiplas páginas.
Exemplo no HTML: 

<link rel="stylesheet" href="style.css">


CSS Interno:

O CSS é escrito dentro da tag <style> no <head> do documento HTML.
Vantagem: Útil para estilos específicos de uma única página.
Exemplo no HTML:
                
                    <head>
                        <style>
                            body { background-color: lightgray; }
                        </style>
                    </head>


CSS Inline:

O CSS é aplicado diretamente a um elemento HTML usando o atributo style.
Vantagem: Para estilos muito pontuais e específicos de um único elemento.
Desvantagem: Dificulta a manutenção e não é recomendado para estilos complexos ou reutilizáveis.
Exemplo no HTML: 
<p style="color: red; font-weight: bold;">Este texto é vermelho e negrito.</p>


HTML, CSS e JavaScript: A Tríade da Web

É importante lembrar que o HTML raramente trabalha sozinho:

  • HTML: Cuida da estrutura e do conteúdo.
  • CSS: Cuida da estilização e do layout (cores, fontes, espaçamentos, etc.).
  • JavaScript: Cuida da interatividade e do comportamento dinâmico da página (animações, validação de formulários, etc.).

Eles se complementam para criar as páginas web ricas e dinâmicas que usamos todos os dias. Se o HTML é o esqueleto, o CSS é a pele e as roupas, e o JavaScript é o sistema nervoso que dá vida e movimento.
